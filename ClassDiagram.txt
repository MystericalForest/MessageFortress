┌─────────────────────────────────────────────────────────────────────────────┐
│                              DATA FLOW                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ 1. ButtonInput reads 8 buttons → 8-bit value (0-255)                        │
│ 2. AccessControl checks if input matches ACCESS_CODE (253672)               │
│ 3. If unlocked: CodeMapper converts input to display code via CODES[]       │
│ 4. CodeDisplay shows either access prompt or mapped code                    │
│ 5. LockSystem controls LED states based on unlock status                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                            SYSTEM STATES                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ LOCKED (Access Code Required):                                              │
│   - Display: Shows 0000 or current input                                    │
│   - LEDs: Constant red (255 brightness)                                     │
│   - Waiting for ACCESS_CODE (253672)                                        │
│                                                                             │
│ UNLOCKED - LOCKED STATE:                                                    │
│   - Display: Shows mapped codes from CODES array                            │
│   - LEDs: Pulsing red (brightness1/brightness2)                             │
│                                                                             │
│ UNLOCKED - CODE STATE:                                                      │
│   - Display: Shows mapped codes from CODES array                            │
│   - LEDs: Pulsing blue (brightness1/brightness2)                            │
│                                                                             │
